# fridayletter

This is a customized collection.
It was migrated from an early custom project, https://github.com/uidaholib/fridayletter_source

The content is harvested annually from the President's communications web pages, https://www.uidaho.edu/president/communications

Friday Letter items have been transferred in a variety of formats:

- emails exported from system. These are email-html contained in the "email" column of the metadata CSV. Use custom "email" Item type.
- emails exported to PDF. "pdf" Item types.
- web archive captures of the Friday Letter as posted on the page. These are perma.cc links, "record" Item types.
